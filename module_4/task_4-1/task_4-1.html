Group Discussion

  1. Why is object oriented development so important in programming?

    2. Object oriented development allows for modular code, reusability, and better maintenance of large systems. It helps in modeling real-world problems effectively.

  2. What is the AccountType static object used for?

    3. The AccountType object stores predefined constants for account types. This ensures consistency and reduces errors when assigning account types.

  3. Why do we use encapsulation in classes?

    4. Encapsulation protects the internal state of an object by restricting access to it, making the code more robust and easier to maintain.

  4. What happens when we use the "new" command in JavaScript?

    5. The new command creates a new instance of a class and initializes it by calling the constructor function.
